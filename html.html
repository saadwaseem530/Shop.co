CLIENT/SERVER NETWORK SYSTEM

A client/server model uses one or more computers as servers, and the other computers on the network are clients.The server is a high speed computers with large hard disk capacity.It contains the nrtwork operating system,the software required to run the network.

One advantage of client/server network system is its ability to handle very large networks eficintly. Another advantage is the powerful management software that another and controls networks activities.
The major disadvatage of it is the high cost of installation and maintenance.
